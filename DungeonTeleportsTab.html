Description

Dungeon Teleports Tab adds a new tab to the Group Finder window, allowing you to quickly teleport to any dungeon or raid.

Features

  • Adds a dedicated Dungeon Teleports tab to the Group Finder
  • Displays all unlocked dungeon and raid teleports
  • Supports all teleports from every expansion

Default Current Season

By default, the addon opens the Current Season tab (currently Midnight – Season 1).

Expansion Selection

Easily switch between expansions using the menu.
