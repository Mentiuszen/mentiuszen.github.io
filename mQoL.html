Description

MQoL is a modular quality-of-life addon focused on account-wide character overview, UI synchronization, Dungeon Teleports, Weekly Rewards tracking, and targeted improvements across supported World of Warcraft clients.

New Features (v1.2.0 - v1.2.1)

  • Account Overview - Tracks characters, levels, classes, gold, played time, professions, and weekly reward progress across the account.
  • Gold Chart - Adds an interactive wealth history chart with improved grid rendering and stable height handling.
  • Weekly Rewards Preview - Shows Great Vault / Weekly Chest progress from saved addon data, including hover information and status icons.
  • Dungeon Teleports Tab - Adds a Group Finder tab with expansion filters, cooldown display, dungeon artwork, and active dungeon highlighting.
  • Profession Tracking - Saves and displays profession details across supported clients, including Classic Era, BCC, Legion, and Retail.
  • Fast Autoloot - Replaces the default CVAR-based looting flow with a faster custom module.
  • Expanded Edit Mode Support - Enables mQoL_EditMode on Mists of Pandaria Classic with version-aware safeguards.

Features

  • Action Bars synchronization - Keeps action bar settings across all characters.
  • Nameplates synchronization - Keeps nameplate settings across all characters.
  • General settings synchronization - Keeps common interface options such as "Auto Loot" and "My Name".
  • Force Edit Mode profile on login - Selected Edit Mode profile is applied on every login.
  • Raid Profiles Synchronization - Shares saved raid profiles across all characters with full automation.
  • Modules Manager - Lets you control which parts of the addon are active.
  • Improved Option Control - Allows disabling or ignoring options you are not interested in.
  • Modern addon interface - Provides a refreshed, consistent UI for addon settings and panels.

Optional Features

  • Mailbox side panel - Adds a mailbox side panel with a character list and quick-send buttons with item categories.
  • Automatic mail subject - Allows setting a default subject for mail.
  • Improved Consolidated Buffs (Mists of Pandaria) - Enhances the default consolidated buffs look.
  • Blizzard bug fixes on Mists of Pandaria Classic - Fixes multiple interface issues, including:
    • Missing PvP reward display.
    • Broken map navigation on Pandaria.
    • Incorrect World Boss display on the World Map.

Planned / Future Updates
